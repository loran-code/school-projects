jQuery combined with CSS gridpane

24-3-2020

It is possible to utilize the jQuery library in two different ways. The first way is to reffer to a CDN (Content Delivery Network), like Google or Microsoft. Here the CDN from the official jQuery website is being used. The url is "https://code.jquery.com/jquery-3.4.1.js" and it can be implemented by placing it in between the html "script" tags like so:

<script src="https://code.jquery.com/jquery-3.4.1.js"></script>

The second option is to download the jQuery library and add it to your map location where the other web development files are located. The benefit of doing so is that you can test your code even when you have no internet access. After downloading the library it can be implemented like so:

<script src="/JS/Jquery.js"></script>
